Overview

This is the base library for most of our programs. It allows you to speed up writing OOP code.

Install

Enter the following command into the terminal in the root folder of the project:

git submodule add https://github.com/Farmat-organization/FSFoundation.git

After that, in the Xcode project settings, find "Build Settings", enter "Header Search Paths" in the search field, add "FSFoundation/FSFoundation".
Under "Build Phases" find "Link Binary With Libraries" and add "libFSFoundation.dylib"

Using

Paste the code below to import the entire library

#include <FSFoundation.hpp>

FSFile MyFSFile;
FSWeb MyFSWeb;
FSArray MyFSArray;

FSFile

To create a string for the user's folder address, use:

std::string Path = MyFSFile.FSPrePath();

Checking the presence of a file or folder at:

bool IsThere = MyFSFile.FSCheckFileOrFolderExists(Path);

Creating a directory:

MyFSFile.FSCreatingDir(Path);

Creating a file:

MyFSFile.FSCreatingFile(Path);

FSWeb

Getting the current time in UTC:

std::string Time = MyFSWeb.FSGetCurrentTimeUTC();

FSArray

Sorting array:

int Array[] = {5, 2, 8, 1, 9};
int Size = sizeof(arr) / sizeof(arr[0]);
bool BigToSmall = true;

MyFSWeb.FSArraySort(Array, Size, BigToSmall);

Outputting an array to the console:

int Array[] = {5, 2, 8, 1, 9};
int Size = sizeof(arr) / sizeof(arr[0]);

MyFSWeb.FSArrayPrint(Array, Size);
