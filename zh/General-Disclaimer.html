基础知识 Farmat

  1. 用户
    1. 用户承诺阅读并同意此文本
    2. 对我们的信息置之不理超过 48 小时就等于终止对话
    3. 用户有责任阅读本文
    4. 我们不对用户的行为负责
    5. 如果我们认为用户违反了属于国际法主体的任何法律，我们没有义务采取法律行动
  2. 条款
    1. 国家 - Farmat 计划参与者或其法定代表所在的任何国家
    2. 我们 - Farmat 团队
    3. 用户 - 使用我们创造的产品、与我们交流或提及我们的人
  3. 我们的风格
    1. 正文
      1. 语言 - 联合国六种正式语言中的任何一种（阿拉伯语、中文、英语、俄语、法语、西班牙语）
      2. 倒装逗号 - 根据通信所用语言的规范（见 『表 1 』或 https://en.wikipedia.org/wiki/Quotation_mark）
      3. 标点符号符合语言规范
      4. 清点应按字母顺序用清点所用的语言或英语进行
    2. 代码
      1. 缩进样式 - Allman、K&R：1TBS、BSD KNF 或 Stroustrup
      2. 编程语言 - C、C++、Objective-C、Swift、Kotlin、Python 或 Java
      3. 在其他情况下，优先选择 Apple 选项 (https://opensource.apple.com/source/Kerberos/Kerberos-75.10.6/KerberosFramework/Kerberos5/Documentation/coding-style)
    3. 计量单位、名称、代码
      1. 国家代码 - ISO 3166
      2. 货币 - ISO 4217
      3. 日期和时间 - ISO 8601
      4. 十进制数 - 手册 BISM SI
      5. 电话号码 - TU-T E.123
      6. 语言代码 - ISO 639
    4. 除非上文有所描述，否则我们建议使用链接中描述的样式： https://help.apple.com/pdf/applestyleguide/en_US/apple-style-guide.pdf
    5. 如果没有，我们建议尽量避免使用这种形式或不使用正式文体
  4. 我们的行动
    1. 我们的任何行为都可能是玩笑或恶作剧
    2. 如果我们的行动或声明可能会有不同的解释，我们将承诺在执法机关提出要求时，或在我们认为有必要的情况下，在任何人提出要求时作出解释
    3. 除执法机构外，我们没有义务向任何人报告我们的行为或关于我们的情况
    4. 代表 Farmat 就任何主题发表的任何声明均不构成 Farmat 成员的观点
    5. Farmat 对政治、宗教等问题不发表任何意见。任何有关政治、宗教等的言论都不反映 Farmat 的观点，但我们认为，在特定条件下，这些言论能达到最佳效果
    6. Farmat 的任何言论都不一定是事实。我们并不声称我们的判断是正确的
    7. 如果我们的行动违反了联合国文件或国家法律，我们将被追究责任
      1. 如果我们开庭审理，法庭承诺允许我们聘请律师
  5. 联系方式
    1. 我们只能通过 Farmat 管理人员或代表我们或我们利益的人员与我们联系
    2. 可以通过我们网站主页 (farmat-team.github.io) 上链接的任何服务进行通信
      1. 如果我们认为合适，我们不必回答你的问题
    3. 什么是 Farmat？
      1. Farmat 是一个国际团队。任何与国家的联系都是不正确的。如果需要联系，Farmat 选择列支敦士登
      2. 点击此处可访问我们所有的社交媒体账户：https://farmat-team.github.io/ru/social_networks.html
      3. 加入我们的行列须遵守以下条款和条件（可能有例外情况）
        1. 位于日光层内
        2. 至少掌握一种联合国官方语言，达到 B2 或以上水平
        3. 作为一个有生命的有机体
        4. 拥有电子邮件帐户
        5. 达到居住国的法定年龄
